## 가상환경 설정 
1. 생성
- global 하게 (어떤 폴더 에서든) 파이썬을 실행할수 있게 *전역설정*
2. 활성화
- 독립적인 폴더 안에서만 활성화 시킬수 있도록 한 것 
```bash
source venv/Scripts/activate
```
- 비활성화
```bash 
source venv/Scripts/deactivate
```

*주피터랩은 주피터 노트북의 확장판이라고 생각하면 됨
